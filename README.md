# Задание для отбора на стажировку в JetBrains летом 2021.

## Что делает приложение.
Приложение очень простое.
Средствами Win32 создаётся окно приложения.
Это окно заполняется цветом #25854b.
Пользователь может, кликая и удерживая левую кнопку мыши, создать и перемещать треугольник.
Когда он отпустит кнопку, треугольник пропадёт.

## Сборка.
* При написании использовался компилятор MSVC 19.28.29913 поставляемый с VisualStudio Community 2019 версии 16.9.3
* Использовался Windows 10 SDK 10.0.18362.0
* Также для сборки требуется CMake версии не ниже 3.14.
* Потребуется установленная библиотека Direct2D, посавляется в скомпилированном виде вместе с указанным релизом VisualStudio.
Сборка осуществляется Release и Debug вариантах. Для указания нужого вариана используется флаг
```-DCMAKE_BUILD_TYPE``` со значением Release или Debug соответственно.
При сборке в режиме Release логирование отключено.
